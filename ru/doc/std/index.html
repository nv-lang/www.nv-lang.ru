Стандартная библиотека

Справочник задокументированных модулей Nova stdlib, сгенерированный через nova doc.

Документация генерируется из исходников командой nova doc. Полные исходники: github.com/nv-lang/nova/tree/main/std. Страницы отдельных модулей — на английском (API reference).

Коллекции

  • []T

    collections.vec

    Функциональные комбинаторы для массивов: map, filter, fold, any, all, first, last.

    Справочник →
  • KV

    collections.hashmap

    Хеш-таблица с открытой адресацией и линейным пробированием. Поддерживает синтаксис map-literal ["k": v, ...], get, insert, remove, iter, filter и другие.

    Справочник →

  • SV

    data.semver

    Semantic Versioning 2.0.0 — парсинг, сравнение и сериализация версий MAJOR.MINOR.PATCH[-pre][+build].
